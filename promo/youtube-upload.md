## Description
Your gamepad is now the remote. 🎮

Couch Controller is a free, open-source Chrome extension that turns any game
controller — Xbox, PlayStation, Switch Pro, or most USB/Bluetooth pads — into a
TV-style remote for your browser. Play and pause videos, change the volume,
seek, switch tabs, scroll pages, and click things with a virtual cursor,
all without touching the mouse.

✨ Features
• Works with any controller Chrome's Gamepad API recognizes
• Every button is remappable — including two-button combos (LB+RB → new tab)
• Bind buttons to any keyboard key, great for site shortcuts
• Virtual cursor + on-screen keyboard driven entirely by the controller
• Open a favorite site (like netflix.com) with a single button
• Settings sync with your Chrome profile
